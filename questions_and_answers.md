###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>info.favoriteFood = "🍝" will not update the food array because food[0] and favourite food objects are diffrent things.The food array is a reference to a separate object in memory than the info object.</i>

</p>
</details>
